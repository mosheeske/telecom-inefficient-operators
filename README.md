# Telecom Operator Performance Analysis

## Business Context
CallMeMaybe is a telecommunications company handling large volumes of inbound and outbound calls. 
Operator performance directly impacts customer experience, waiting times, and service quality.

## Objective
Identify inefficient call center operators using operational KPIs and statistical analysis in order to support performance monitoring and improvement actions.

## Tools Used
- Python (pandas, numpy, scipy)
- Jupyter Notebook
- Statistical testing (Mannâ€“Whitney U)
- Data visualization

## Methodology
- Data cleaning and preprocessing
- KPI calculation (missed calls rate, average waiting time)
- Group comparison using statistical hypothesis testing
- Operator ranking based on performance indicators

## Key Findings
- Significant differences in waiting times between operator groups
- A subset of operators consistently shows higher missed-call rates
- Results support targeted supervision and training initiatives

## Outcome
The analysis provides a data-driven basis for identifying underperforming operators and improving overall call center efficiency.
